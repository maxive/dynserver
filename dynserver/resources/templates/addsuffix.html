Manage zones

  • Add zone
  • Configured zones

Add new zones, in which hostnames can be defined ($ORIGIN). When specifying a new hostname, the user can choose in which zone it should be. By adding a zone example.org, a user can create a hostname like myhost.example.org.

Be aware, that no duplicates are allowed! Specifying a zone here means, users are allowed to add any RFC-compilant subdomain (i.e. mail) to it. None of the subdomains may appear in your regular powerdns database!
